# Kmart-sales-dashboard
**Project Overview**
An interactive Excel dashboard created to analyse Kmart’s multi-year sales data across product categories, SKUs, and regions. This dashboard enables clear visibility into sales trends, category performance, and regional insights for better retail decision-making.
**Key Features**
Sales Trend Analysis — Line & bar charts showing monthly and yearly sales movement
Category & SKU Performance — Pivot tables and slicers to filter categories, subcategories & top-selling products
Regional Insights — Region/state-wise sales comparison using pivot charts & map visuals
Interactivity — Slicers, dropdowns, and drill-down filters
KPI Summary — Total Sales, AOV, Units Sold, YoY Growth
**Techniques & Tools**
Excel (Pivot Tables, Pivot Charts, Slicers)
SUMIFS, COUNTIFS, INDEX/MATCH, IFERROR
Data Cleaning & Preparation
Conditional Formatting
Drill-Down & Slice-and-Dice Analysis
**Business Impact**
This dashboard simulates a real retail analytics environment where large datasets can be easily explored to identify trends, monitor category performance, compare regions, and support strategic decision-making for retail teams.
